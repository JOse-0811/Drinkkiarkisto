## KUVAUS
Drinkkiarkisto on verkkosivusto, jonka tarkoituksena on auttaa käyttäjiä löytämään helposti uusia ja tuttuja juomareseptejä. Käyttäjät voivat myös ehdottaa omia reseptejä, jotka ylläpitäjä hyväksyy ennen julkaisua. 
## MITÄ OPIN
- Käyttäjien salasanojen turvallinen käsittely
- PHP:n perusteet ja backend-logiikka
- Verkkosivun toteutus alusta loppuun
## TEGNOLOGIA
- HTML
- PHP
- JavaScript
- CSS
## KÄYTTÖ
Avaa kirjaudu.php tai rekisteri.php selaimessa.
## HUOMIO
Tämä projekti on tehty koulutyönä toisen vuoden loppuprojektina.
